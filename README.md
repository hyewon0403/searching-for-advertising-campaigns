## 광고 캠페인 조회 기능 기획

### 프로젝트 소개
산재되어 있던 데이터를 구조화하기 위해 데이터베이스를 구축하고, 수집된 데이터를 전 직원이 쉽게 업무에 활용할 수 있도록 광고 캠페인 조회 기능을 기획했습니다.
이를 통해 과거 광고 캠페인 이력과 함께 주로 집행하는 광고 상품 정보를 제공했습니다.
